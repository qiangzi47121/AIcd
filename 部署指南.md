# 🚀 Vercel 免费部署指南

> 部署到 Vercel 后，任何设备（手机/电脑）都能通过公网地址访问！

---

## 📋 部署步骤（5分钟完成）

### 第一步：下载项目包
找到项目文件夹里的 **`ai-radio-app.zip`** 文件，解压到任意位置。

### 第二步：打开 Vercel 并登录
1. 浏览器打开 👉 **https://vercel.com**
2. 用 GitHub 账号登录（没有就去 github.com 免费注册一个）
3. 点击 **"Add New..."** → **"Project"**

### 第三步：上传代码
点击 **"Import Third-Party Git Repository"** 下方的 **"Or create a new Git repository"**，按提示操作：
- 在 GitHub 上新建一个仓库，命名为 `ai-radio-app`
- 把解压后的所有文件上传到 GitHub 仓库

或者更简单的方式（推荐）：
1. 访问 👉 **https://vercel.com/new**
2. 选择 **"Deploy"**
3. GitHub 会弹出授权窗口，授权后选择刚才创建的仓库

### 第四步：配置项目
在 Vercel 配置页面：
- **Framework Preset**: `Other`
- **Root Directory**: `./`（保持默认）
- 其他不用改，直接点 **"Deploy"**

### 第五步：等待部署完成 ⏱️
Vercel 自动构建，约 30-60 秒后获得一个 `.vercel.app` 的公网地址，例如：
```
https://ai-radio-app.vercel.app
```

---

## 📱 手机访问
直接用手机浏览器打开上面获得的地址，例如：
```
https://ai-radio-app-xxx.vercel.app
```
点击浏览器菜单 → **"添加到主屏幕"**，完成！

---

## 🔧 自定义域名（可选）
在 Vercel 项目设置 → Domains，可以绑定自己的域名。

## ❓ 没有 GitHub 怎么办？

### 方案 A：用 GitHub Desktop（最简单）
1. 下载 [GitHub Desktop](https://desktop.github.com/) 并安装
2. 登录 GitHub 账号
3. 点击 **"Add an Existing Repository"**
4. 选择 `ai-radio-app` 文件夹
5. 点击 **"Publish repository"**
6. 去 Vercel 导入这个仓库

### 方案 B：命令行安装 Git
```bash
# 下载 Git for Windows
winget install --id Git.Git -e --source winget

# 然后在项目目录打开 PowerShell：
git init
git add .
git commit -m "AI Radio App"
# 按提示登录 GitHub
git remote add origin https://github.com/你的用户名/ai-radio-app.git
git push -u origin main
```

### 方案 C：Netlify 直传（无需 Git）
1. 打开 👉 **https://app.netlify.com/drop**
2. 把 `public` 文件夹拖进去
3. 获得 Netlify 提供的临时地址
4. （注意：Netlify 免费版不支持 Serverless Functions，AI播报功能需要在本地运行）

## ⚠️ 注意事项

- **免费版 Vercel** 不支持长连接（WebSocket），但音乐播放、AI播报完全够用
- **Jamendo API** 在中国大陆访问可能较慢，已内置 fallback 数据
- 首次访问需要联网，之后 PWA 可以离线使用基础功能
